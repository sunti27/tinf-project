Introduction

This is a tutorial on how to make a discord bot in the programming language python.

Getting started

  1. Creating a discord bot application

    2. First of all, go to the Discord developer website. There you need to login with your Discord account, if you haven't already. The next step is to create a New Application You will find that button in the top right corner. Then you have to enter the name of your bot/application and click Create

    After this you have to add a bot to your application, to do this select it, then go into the menu "Bot", which you find on the left side of the page. Then click Add Bot and confirm the action.

  2. Authorize your bot

    3. The next step is to authorize your bot/add it to a server, to do this you have to replace ID_OF_YOUR_BOT with the client id of your bot, which you an find in the menu "General Information", in the following url. After this you just have to copy and paste it into your browser.

    https://discordapp.com/api/oauth2/authorize?client_id=<ID_OF_YOUR_BOT>&scope=bot&permissions=1

    On the website that appears you have to add one of your servers via the dropdown menu, then you just click Authorize and it should magically appear on that server.

    Now you just have to give your bot the permissions he needs for his job, which you can do on your discord server threw the role that gets created by adding him. Or you can do it the fancy way, while you are adding it by creating a permission integer in the "Bot" menu on the developer portal and replacing the 1 at the end of the url with it.

  3. Installing Python

    4. Now you have to install python, you can also use another programming lanuguage, since you probably already read the title, you know that this site is focusing on python.

    You can find the latest version of python here, on the site just click download button and run the downloaded executeable.

    A window like this should open. Before you proceed make sure you tick the checkbox next to ADD Python *.* to PATH and then you are free to proceed clicking INSTALL NOW.

    You might have to validate your action by clicking OK, after this you just have to wait.

    When the installer finished his job you should see the following window. Now you are nearly ready to go.

  4. Install the discord library for python

    5. The final step before you are able to start coding, is the installation of the discord library.

Basics

  1. Creating a basic bot
    2. Now we can start programming the bot. First, we have to import the installed discord libraries.
    This will look like this: 
    main.py
    import discordfrom discord.ext import commands
    Now we've got access to all of the required modules. After doing this we create an object from the bot class. 
    main.py
    bot = commands.Bot(command_prefix="<your_prefix>")
    We are now able to add commands and events. An important event is the on_ready event. Lets add this to our code. 
    main.py
    @bot.listen("on_ready")
                            def on_ready_event():
                                print(f"Logged in as: {bot.user}")
    In the end, we need one more line: 
    main.py
    bot.run(<your_token>)
    After doing all of this you can run your file, which looks like this: (prefix and token are just examples) 
    main.py
    import discord
                            from discord.ext import commands
                            
                            bot = commands.Bot(command_prefix="!")
                            
                            @bot.listen("on_ready")
                            def on_ready_event():
                                print(f"Logged in as: {bot.user}")
                             bot.run("NDgxMDk2NTUxOAA5MTU1MDg4.Xdp6PA.dcE076j1NnNhk7gfnlgeF0txN-4")

    You can run the file by either running it in your ide or in the console by typing:

    cd /path/to/directory

    Windows: python main.py

    MacOS and Linux: python3 main.py

    Congratulation, you created your first discord bot :)
